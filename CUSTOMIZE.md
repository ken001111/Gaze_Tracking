# Customization Checklist

Before publishing to GitHub, please update the following with your information:

## Files to Update

### 1. LICENSE
- [ ] Replace `[Your Name]` with your actual name
- [ ] Update year if needed (currently 2026)

### 2. AUTHORS.md
- [ ] Replace `[Your Name]` with your actual name
- [ ] Update email address
- [ ] Update GitHub username
- [ ] Add any co-authors or contributors

### 3. README.md
- [ ] Update citation section with your name
- [ ] Update GitHub URL in citation: `https://github.com/yourusername/gaze-tracking`
- [ ] Add your GitHub username to badges (optional)

### 4. GITHUB_SETUP.md
- [ ] Update repository name suggestions if needed
- [ ] Update remote URL examples with your username

### 5. README_ENHANCED.md (if keeping)
- [ ] Update any references to your name/institution

## Optional Customizations

### Repository Name Suggestions
- `stanford-gaze-tracking`
- `gaze-tracking-system`
- `eye-tracking-research`
- `clinical-gaze-tracking`

### Add Your Information
- Add your photo/avatar to AUTHORS.md
- Add project logo/banner to README
- Add screenshots of the GUI
- Add demo video link

### Repository Description
Suggested: "Enhanced real-time gaze tracking system for clinical research - Stanford Neuroradiology Project"

## Quick Find & Replace

Search for these placeholders and replace:
- `[Your Name]` → Your actual name
- `yourusername` → Your GitHub username
- `your.email@stanford.edu` → Your email
- `your-repo-url` → Your actual repository URL

## After Customization

1. Review all files
2. Test that everything works
3. Follow GITHUB_SETUP.md to publish
4. Update repository description on GitHub
5. Add topics/tags on GitHub
